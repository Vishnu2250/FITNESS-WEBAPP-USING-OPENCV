#  Fitness & Yoga Trainer 🧘‍♂️💪

AI Fitness & Yoga Trainer is a Flask-based web application that uses Computer Vision and MediaPipe to detect body posture and track exercises and yoga poses in real time through a webcam. The application helps users perform workouts correctly by analyzing body movements and posture using AI.

## ✨ Features

### 🏋️ Exercise Detection

* Biceps Curl Counter
* Push-Ups Counter
* Triceps Exercise Tracking
* Squats Counter

### 🧘 Yoga Pose Detection

* T-Pose Detection
* Tree Pose Detection
* Warrior Pose Detection

### 📊 Additional Features

* BMI Calculator
* Real-time webcam streaming
* Pose tracking using MediaPipe
* Exercise repetition counting
* Interactive web interface using Flask
